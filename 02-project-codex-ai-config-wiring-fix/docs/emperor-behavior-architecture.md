# 皇帝行为、心情与裁断硬规则（架构目录版）

## 一、文档定位

本文件定义皇帝心情、真心值、独寝、侍寝反馈与养心殿裁断。  
系统负责真实结果，AI 只补充对白与场景文案。

## 二、核心状态

皇帝至少维护以下字段：

- `mood`：心情，`-100~100`
- `trueHeart[consortId]`：对每位妃嫔的隐藏真心值，`-100~100`
- `lastNightInterest[consortId]`
- `highestTrueHeartConsortId`
- `rollingDeathCount6m`

## 三、心情分段

- 悲痛：`-100~-50`
- 悲伤：`-49~0`
- 低落：`1~20`
- 平常：`21~50`
- 愉悦：`51~70`
- 喜悦：`71~100`

## 四、心情月恢复

若当月无人死亡：

- `mood < 0`：月末 `+8`
- `mood >= 0`：月末 `+3`

若滚动半年后宫死亡人数首次达到 `6` 人及以上：

- `mood = min(mood, -50)`

## 五、独寝率

- 悲痛：`50%`
- 悲伤：`35%`
- 低落：`25%`
- 平常：`15%`
- 愉悦：`10%`
- 喜悦：`5%`

## 六、真心值

### 6.1 开局范围

- 兰因絮果：`20~50`
- 浮生如梦：`0~60`
- 影落掖庭：`0~40`
- 尘缘夙错：`-20~20`

### 6.2 真心值作用

- 夜晚互动额外加权
- 御书房带回概率
- 养心殿求情偏向
- 真爱值最高保护

并列最高时，按：

1. 宠爱更高
2. 位分更高
3. 入宫更早

## 七、侍寝反馈

### 7.1 兴致结算

- `< 40`
  - 心情 `-3`
  - 宠爱 `-2`
  - 真心值 `-1`
- `40~69`
  - 心情 `+3`
  - 宠爱 `+2`
  - 真心值 `+1`
- `70~89`
  - 心情 `+5`
  - 宠爱 `+4`
  - 真心值 `+2`
- `90~99`
  - 心情 `+7`
  - 宠爱 `+6`
  - 真心值 `+4`
- `100`
  - 心情 `+10`
  - 宠爱 `+8`
  - 真心值 `+6`
  - 额外声望 `+10`

### 7.2 第三方美言与抹黑

玩家侍寝时可：

- 为交好妃子美言：目标宠爱 `+3~5`
- 对交恶妃子诉苦 / 抹黑：目标宠爱 `-3~5`

若其他妃子侍寝：

- 与玩家交好：`80%` 概率为玩家美言，玩家宠爱 `+3~5`
- 与玩家交恶：`20%` 概率抹黑玩家，玩家宠爱 `-3~5`

以上均不算主动宫斗，不扣福德，不进调查。

## 八、孕期陪伴

怀孕角色不再侍寝，只进入陪伴池。  
陪伴结算：

- 心情 `+3`
- 宠爱 `+2`
- 真心值 `+2`

若其宠爱 `>= 60`，额外心情 `+1`。  
若其为当前真心值最高对象，额外真心 `+1`。

## 九、流产、生育、死亡对皇帝心情

- 自然流产：心情 `-12`
- 宫斗导致流产：按宫斗重负面处理
- 死胎：心情 `-18`
- 妃嫔死亡：按死亡事件处理

## 十、日间行为与互动

### 10.1 行动范围

- 清晨：正阳门，可等待下朝并按概率偶遇皇帝，成功宠爱 `+1`。
- 上午：养心殿处理政务，可求见但成功率较低。
- 中午 / 下午 / 傍晚：养心殿可求见，御花园 / 建章宫可按行动表概率偶遇皇帝。
- 夜晚 / 深夜：养心殿不接受后宫求见，由内侍劝归。

地点进入本身已经消耗时间和体力；求见、等待下朝、公共偶遇和皇帝互动结束不再额外推进第二格。

### 10.2 互动收束

- 皇帝交互每次只允许 `1` 次主行动，另可在送礼 / 美言 / 诉苦中选 `1` 次附加话题。
- 主行动结果必须先完整播放，之后才进入告退或恭送收束。
- 公共地点偶遇使用外景口径：随驾内侍催请、玩家退到一旁恭送；不得复用殿内求见的退出文案，也不得主行动后直接黑屏。
- 养心殿求见使用殿内口径：内侍提醒政务时辰，玩家依礼退出养心殿。

## 十一、养心殿裁断

### 11.1 养心殿说服值

```txt
养心殿说服值 =
20
+ 玩家宠爱 / 2
+ (玩家容貌 + 玩家气质) / 60
+ max(0, 玩家真心值) / 8
+ 皇帝心情修正
- 事件修正
```

心情修正：

- `mood < 0`：`-10`
- `0~20`：`-5`
- `21~50`：`0`
- `51~70`：`+5`
- `71~100`：`+10`

事件修正：

- 轻：`5`
- 中：`10`
- 重：`15`

### 11.2 裁断效果

- 说服值高：降低获罪率或降低惩罚
- 说服值低：维持原判，重案可能额外加压

## 十二、统一语义

- 皇帝心情影响独寝、册封和养心殿偏向
- 真心值影响夜晚权重、御书房、求情与保护
- 侍寝中的美言 / 抹黑独立于宫斗系统
- 养心殿只修正裁断，不重算整起案件
