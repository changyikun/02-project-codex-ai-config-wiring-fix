# 归档文档说明

本目录保存历史设计稿、旧规则稿和当前版本不再接入的接口说明。

归档文档的规则：

- 不再随日常开发同步维护。
- 不作为当前实现或策划裁决的权威来源。
- 与 `docs/README.md` 中列出的当前维护文档冲突时，以当前维护文档为准。
- 若需要重新启用，先移出 `docs/archive/`，再登记到 `docs/README.md` 的当前维护文档表中。

## 分区

| 目录 | 内容 |
| --- | --- |
| `legacy-rules/` | 旧硬规则稿、自然语言说明稿、旧 B 版宫斗规则 |
| `legacy-ai/` | 当前玩法未接入的 AI / Foundation / 自定义剧情妃接口旧说明 |
| `proposals/` | persona、世界记忆等历史 proposal |
| `retired-overviews/` | 过度膨胀、已经无法作为维护入口的总交接 / 总规则文档 |
