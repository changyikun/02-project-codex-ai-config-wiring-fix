# 经济、宫务开销与案件银两干预硬规则（架构目录版）

## 一、文档定位

本文件定义银两来源、月俸结算、宫务开销、家族与朝堂运作、子嗣支出，以及案件中的银两干预规则。  
所有真实结果都由系统结算，AI 只补充文案。

## 二、核心原则

- 固定月度开销由玩家选择的用度策略决定，避免开局性格选项直接变成一次性数值奖励。
- 真正的大额消耗留给朝堂、家族、子嗣、节庆和夺位事件。
- 高宠必须永远是净收益，不允许出现“越受宠越亏钱”。

## 三、月俸与赠银

### 3.1 月俸

玩家每月按当前位分领取月俸，月俸基表沿用位分系统。

### 3.2 克扣与高宠补贴

- `位分 < 妃` 且 `宠爱 1~19`：月俸按 `70%` 发放
- `位分 < 妃` 且 `宠爱 <= 0`：月俸按 `50%` 发放
- 冷宫：月俸为 `0`
- `宠爱 >= 61`：当月实际月俸 `+20%`

### 3.3 倾情妃赠银

- 条件：`倾情 > 60`
- 每月最多 `2` 名妃子赠银
- 每名赠送自己当月**实际月俸**的 `1/4`
- 赠银总额允许超过玩家自身基础月俸

## 四、固定月度开销

### 4.1 月度用度

设玩家当月实际月俸为 `S`。

```txt
节衣缩食 = floor(S * 0.25)，声望 -5，健康 -1
量入为出 = floor(S * 0.5)，声望 0，健康 0
锦衣玉食 = floor(S * 0.75)，声望 +10，健康 +1
```

开局三选项对应本月用度策略；寝殿“调整用度”只能登记下月策略，跨月结算后生效。开场和寝殿调整入口都保留第四个说明选项，只解释三档含义，不写入 `monthlyExpenseStrategy` 或 `nextMonthlyExpenseStrategy`。

### 4.2 子嗣月抚养

- `0~35` 月：每名子嗣每月 `20`
- `36` 月及以上：每名子嗣每月 `35`

### 4.3 子嗣礼仪开销

- 满月礼：`180`
- 周岁礼：`260`
- 两岁礼：`320`

若不支付：

- 玩家声望 `-15`
- 对应子嗣宠爱 `-3`

## 五、礼物、补品与珍稀度

### 5.1 绣品 / 字画 / 补品品质表

| 品质 | 颜色 | 单价 | 送礼好感 | 补品健康效果 | 补品容貌效果 |
| --- | --- | --- | --- | --- | --- |
| 绿色 | Green | `70` | `+5` | `+20` | `+5` |
| 蓝色 | Blue | `130` | `+10` | `+40` | `+10` |
| 紫色 | Purple | `180` | `+15` | `+60` | `+15` |
| 红色 | Red | `250` | `+20` | `+80` | `+20` |

规则：

- 绣品、字画按品质增加目标好感，不提供直接数值恢复。
- 补品可送礼，也可自用。
- 补品送礼时：目标好感按品质增加，同时目标健康与容貌按品质恢复。
- 补品自用时：不增加好感，但恢复玩家自身健康与容貌。

### 5.2 后期容貌 / 气质成长行为

- 习舞奏乐：`-2` 体力，`1` 时间格，`乐理 +2`，`气质 +3`，`容貌 +3`
- 御花园闲逛：`-1` 体力，`1` 时间格，`气质 +3`
- 华清池温泉：`-1` 体力，`1` 时间格，`容貌 +3`

以上为稳定成长项，可与送礼、侍寝、剧情奖励并行存在。

### 5.3 其他常规礼物

- 御膳房珍馐：`60`

### 5.4 稀有礼物与丹药池

- 稀有首饰 / 异域珍玩 / 珍贵药材：`260`
- 特殊庆贺级礼物：`420`

杜娘的稀有池包含：

| 物品 | 单价 | 效果 |
| --- | --- | --- |
| 冷香丸 | `500` | `气质 +100` |
| 驻颜丹 | `500` | `容貌 +100` |
| 延寿丹 | `500` | `寿命 +5` |
| 缠梦香 | `1000` | 赠送给目标后，每旬额外 `寿命 -0.6`，可叠加 |

规则：

- 每次杜娘刷新时，上述每种稀有物品独立以 `40%` 概率出现。
- 同一种稀有物品单次刷新库存不超过 `2` 个。
- 普通礼物与普通补品不受该 `40%` 稀有刷新限制。
- 若玩家 `药理 > 80`，可花费 `500` 银两自行制作缠梦香，每次 `70%` 概率成功。

### 5.5 赏赐与特殊赠礼

- 任意妃子若 `倾情 >= 80`，首次达到阈值时会赠送玩家 `冷香丸 / 驻颜丹` 其中之一。
- 任意高宠爱妃子若 `宠爱 >= 61`，每月有 `60%` 概率获得皇帝赏赐。
- 任意高太后好感妃子若 `太后好感 >= 61`，每月有 `60%` 概率获得太后赏赐。

赏赐池建议：

- 皇帝赏赐：银两、普通礼物、高品质补品、冷香丸、驻颜丹
- 太后赏赐：银两、绣品、字画、补品、礼佛物件

限制：

- 皇帝赏赐池中不出现 `延寿丹 / 缠梦香`
- 太后赏赐池中不出现 `冷香丸 / 驻颜丹 / 延寿丹 / 缠梦香`
- 缠梦香不进入自然赏赐池，只能来自杜娘或玩家自制

## 六、朝堂与家族运作

### 6.1 朝臣打点

- `政治 < 20`：不可正式朝臣打点
- `政治 20~39`
  - `180` 银两：朝臣支持 `+2`
  - `320` 银两：朝臣支持 `+4`
- `政治 40~59`
  - `180` 银两：朝臣支持 `+3`
  - `320` 银两：朝臣支持 `+5`
  - `500` 银两：朝臣支持 `+8`
- `政治 >= 60`
  - `180` 银两：朝臣支持 `+3`
  - `320` 银两：朝臣支持 `+6`
  - `500` 银两：朝臣支持 `+10`

### 6.2 父亲官职阶梯

父亲官职决定玩家每月固定声望补贴：

- `0级` 商贾 / 罪臣：`+0`
- `1级` 六品：`+2`
- `2级` 四品：`+4`
- `3级` 二三品：`+6`
- `4级` 一品：`+8`
- `5级` 国公 / 镇国公：`+10`

同时对自然声望有家世修正：

- 商贾之女：每月自然声望额外 `-3`
- 罪臣之后：每月自然声望额外 `-5`
- 六品：`0`
- 四品：`+1`
- 二三品 / 一品：`+2`
- 镇国公：`+3`
- 和亲公主 / 外族线：`+1`

### 6.3 家族稳定度

入宫前 `6` 个月，父亲官职不会降位。  
第 `7` 个月起，每季度结算一次家族安全值：

```txt
家族安全值 =
22
+ 政治 * 0.25
+ 朝堂衷心度 * 0.45
+ 家族接济加值
+ 官员游说加值
- 树敌压力
```

V1 中树敌压力可先按下式实现：

```txt
树敌压力 =
max(0, 主动宫斗次数 - 2) * 3
+ max(0, 45 - 宠爱) / 4
+ 事件型进冷宫次数 * 8
```

### 6.4 家族接济

当前版本只保留第一档：

- `120` 银两：家族接济
- 效果：`家族接济加值 +10`，季度额外声望 `+12`

### 6.5 父亲官职波动

季度结算时：

```txt
父亲降位概率 = clamp(0, 70, 45 - 家族安全值)
父亲恢复概率 = clamp(0, 65, 家族安全值 - 38)
```

一次性降位扣声望：

- `1级`：`-50`
- `2级`：`-90`
- `3级`：`-130`
- `4级`：`-180`
- `5级`：`-240`

注意：

- 降位扣声望是**一次性**
- 降位后，新的官职等级仍会继续每月给玩家对应声望补贴

## 七、节庆与大额事件

### 7.1 年度节庆

持有 `协理六宫` 的妃子、皇贵妃、皇后，每年都要处理一次节庆置办。

- 费用：`260`
- 置办后：
  - 其他妃子野心 `-4`
  - 玩家压力 `-10`
  - 玩家野心 `+6`
  - 玩家声望 `+20`
- 若不置办：
  - 玩家声望 `-35`
  - 玩家压力 `+8`
  - 其他妃子野心 `+2`

### 7.2 后期夺位大事件

用于 `唯我独尊 / 权力巅峰` 等结局线：

- `2000`
  - 可获得虎符
  - `60%` 概率以偷取方式拿到
- `4000`
  - 正常游说获得虎符
- `6000`
  - 获得虎符
  - 朝堂衷心度 `+20`
  - 政治 `+15`
  - 皇帝真心值 `+20`
  - 野心额外上升

投入越高，夺位成功率越高。

## 八、案件银两干预

### 8.1 个人事务

当玩家本人卷入调查时，基础银两干预统一为：

- `20` 银两：玩家自身嫌疑 `-5`

若案件存在已锁定的陷害妃嫔，则玩家可改为：

- `20` 银两：陷害妃嫔嫌疑 `+5`

该干预为**可重复操作**，直到：

- 银两不足
- 玩家主动停止
- 对方嫌疑达到 `100`
- 案件结案

若陷害妃嫔嫌疑达到 `100`，案件进入 `pending_verdict`，等待养心殿或内廷裁断；裁断完成后才写入定罪与处罚。

### 8.2 旁观案件

若调查案件与玩家无直接嫌疑关系，纪事仍显示主要嫌疑人。  
玩家可对任意嫌疑人执行：

- `20` 银两：嫌疑 `-5`
- `20` 银两：嫌疑 `+5`

同样可重复操作，直到银两不足或案件结案。

## 九、实现优先级

优先落地以下 6 项：

1. 月俸、克扣、高宠补贴、倾情赠银
2. 按月俸百分比计算的宫务开销
3. 子嗣月抚养与礼仪支出
4. 朝臣打点
5. 家族接济与父亲官职季度波动
6. `20 银两 -> ±5` 的案件干预
