# 晋升、降位、冷宫与协理六宫硬规则（架构目录版）

## 一、文档定位

本文件定义位分、应有位分、月结算升降、冷宫、皇后 / 皇贵妃特殊位、以及协理六宫权限。  
系统负责真实判定，通报与后续剧情只负责呈现，不参与位分判定。

## 二、总原则

- 声望决定 `应有位分`
- 月结算负责把 `实际位分` 向 `应有位分` 推进
- 自然晋升每月最多 `2` 级
- 自然降位每月最多 `2` 级
- “未侍寝不能晋升”正式废止

## 四、月结算推进顺序

1. 先结算当月处罚与声望变化
2. 处理“皇帝真爱值最高”的处罚保底
3. 判定是否进入冷宫
4. 计算应有位分
5. 推进实际位分
6. 若实际位分发生晋升，生成最高优先级太监传旨通报
7. 生成普通月初 / 旬初通报

## 五、晋升与降位

### 5.1 晋升条件

- 声望达到目标位分门槛
- 目标位分有空位
- 皇帝心情允许
- 存活
- 未疯癫
- 当前不在冷宫
- 未受血脉与路线限制

### 5.2 皇帝心情限制

- `心情 < 0`：本月不晋升任何人
- `心情 < 20`：本月不可晋升到 `妃位及以上`
- `心情 < 50`：本月不可新立 `皇后`

### 5.3 同位竞争顺序

当高位空位不足、多人同时满足条件时，按以下顺序择优：

1. 声望更高
2. 宠爱更高
3. 真爱值更高
4. 当前位分更高
5. 入宫时间更早

### 5.4 玩家晋升通报

- 玩家实际位分在月结算中上升时，必须生成独立的“晋封旨意”通报。
- 晋升通报由传旨内侍在清晨播报，优先级高于普通月初 / 旬初通报、侍寝通报和养心殿裁断传唤；所有清晨通报都必须等黑屏淡出结束后才可出现。
- 晋升通报只负责演出旧位分、新位分和迁宫信息；真实位分、声望、住处仍以月结算后的 state 为准。
- 晋升通报使用专用圣旨展开演出：先由内侍在寝殿送旨，玩家确认后进入遮罩圣旨；圣旨从合拢卷轴展开为横幅，两侧卷轴柄部展开后仍保留，且卷轴柄高度必须长于中间画幅以保证视觉衔接。
- 圣旨题签由 `material-202606231107-018.png` 与 `material-202606231107-019.png` 两张字形素材组成，定位在圣旨底图右侧圆形题签区域；不得回退为 CSS / 文本绘制。
- 圣旨正文必须包含通报时间生成的宫历日期，按底图竖线分列，新位分单独成列放大显示，正文排版需避开右侧题签圆纹；长句应按标点拆列，末尾“钦此”等短列不得被推入裁剪边界。

## 六、血脉与路线限制

- 和亲公主 / 异国贡女默认封顶 `皇贵妃`
- 仅 `尘缘夙错` 线路允许其自然进入皇后线
- 生母为和亲公主 / 异国贡女的孩子默认不得立储，仅 `尘缘夙错` 例外

## 七、皇后与皇贵妃

### 7.1 皇后

- 皇后唯一
- 天然拥有六宫管理权

### 7.2 皇贵妃开放条件

仅在以下任一条件成立时允许新设皇贵妃：

- 当前无皇后
- 皇后 `健康 < 400` 且持续 `3` 个月
- 皇后 `宠爱 < 20` 且持续 `6` 个月

### 7.3 皇后恢复后的处理

皇后恢复后，已在位皇贵妃不自动撤位。  
只有在声望不足、被处罚、被废黜或进入冷宫时，皇贵妃才会离开该位。

## 八、冷宫

### 8.1 进入冷宫的三条路径

满足任一条件即进入冷宫：

1. 宫斗 / 调查 / 明旨惩处导致声望 `< 0`
2. 自然声望连续 `2` 个月 `< -50`

### 8.2 真爱值最高保护

若本次受罚角色为当前皇帝真爱值最高妃子，则处罚后声望保底为 `100`。  
该保护只适用于处罚事件，不覆盖自然月度波动。

### 8.3 冷宫中的限制

- 不参与侍寝与陪伴
- 不参与自然晋升
- 不可被授予协理六宫
- 大部分正常后宫事件关闭

### 8.4 冷宫接出

若角色声望重新回到 `0` 或以上，且没有剧情强制冷宫标记，则月结算时自动接出。

## 九、协理六宫

### 9.1 基础规则

- 同时最多 `1` 人
- 授予条件：位分 `>= 妃`、存活、未疯癫、未在冷宫

### 9.2 调位权限

每 `3` 个月可对位分低于自己的妃嫔进行一次调位，最多 `2` 级。

- 若对方与自己好感 `> 60`：可提位
- 若对方与自己交恶：可打压

若玩家被他人提位或打压，声望会自动调整到新位分的最低限额。

### 9.3 年度节庆义务

持有协理六宫、皇贵妃、皇后，每年需处理一次节庆置办：

- 支出 `260`
- 成功置办：
  - 全后宫其他妃子野心 `-4`
  - 玩家压力 `-10`
  - 玩家野心 `+6`
  - 玩家声望 `+20`
- 若不置办：
  - 玩家声望 `-35`
  - 玩家压力 `+8`
  - 全后宫其他妃子野心 `+2`

## 十、统一语义

- 位分主轴始终是声望
- 冷宫不再只有“自然负声望”一种入口
- 皇后恢复后不会自动冲掉已在位皇贵妃
- 协理六宫既是权力，也是稳定的年度银两消耗点
