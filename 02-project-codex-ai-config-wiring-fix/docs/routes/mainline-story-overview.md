# 四条主线故事情节总览

> 文档用途：作为后续主线事件表、剧情文本、触发规则和结局条件的总方向。  
> 范围说明：本文件介绍四条路线的完整故事走向，不替代单条路线的详细节点文档。`影落掖庭` 的详细案件、事件与文本见 `docs/routes/yingluoyeting-mainline.md`。  
> 玩法边界：主线剧情以本地固定文本和硬规则触发为主，不依赖实时 AI 生成剧情。

## 1. 兰因絮果

### 1.1 路线定位

`兰因絮果` 是高起点中宫权力线。玩家以皇后身份入局，开局即住椒房殿，拥有最高名分，却也被名分困住。她不是从低位往上爬，而是在一开始就站在六宫与朝局中央：所有人都看她的体面、手段、宠爱和决断。

这条线的核心问题是：玩家究竟是要做皇帝容安的妻子、后宫秩序的执掌者，还是独立于帝王情爱之外的权力共主。

### 1.2 故事开端

玩家谢令仪是镇国公嫡女，凤冠压顶入主中宫。外人看她尊贵，实际上她面对的是更深的束缚：皇帝旧情未断，六宫妃嫔各有来处，前朝也把她视作镇国公府在宫中的延伸。

开局时，椒房殿表面安稳，实则危机已经铺开。姚玲儿得宠且有表兄妹旧情，柳仪芳因长相或气质近似玩家而被卷入替影风波，江晚晚清醒守密、看透帝王本质却不轻易站队。玩家必须在中宫体面、夫妻情分、妃嫔关系和朝局压力之间做取舍。

### 1.3 主要冲突

第一层冲突是“皇后与宠妃”。姚玲儿高傲、重脸面、与皇帝有旧情，前期把玩家视为威胁。她不是单薄恶妃，而是一个同样被旧情、权位和体面困住的人。玩家可以压制她，也可以在利益或敬重中与她结盟。

第二层冲突是“真身与替影”。柳仪芳对玩家初始有好感，温顺体贴，却也因为自己像玩家而获得如今位置。她的故事不是争宠挑衅，而是从替身位置里找回自我。玩家若只把她当工具，她会始终停在影子里；若承认她的自我，她可成为争宠和后宫经营中的温柔助力。

第三层冲突是“守密与站队”。江晚晚温柔克制，擅写话本却不敢示人，对皇帝并不真正爱慕。她不容易被攻略，也不被普通送礼打动。她的关键在于玩家能否守住秘密、尊重边界，并在权力局中给她一个不必被利用的位置。

第四层冲突是“夫妻还是政治答案”。容安对中宫既有依赖，也有衡量。玩家若只追求独宠，可能失去后宫大局；若只追求权力，夫妻情分会转冷；若两者都要，就必须承受更高的猜忌与压力。

### 1.4 故事阶段

第一阶段：中宫立威。玩家熟悉皇后权限，处理宫宴、请安、选秀、妃嫔冲突，建立自己是“能管事的皇后”还是“只守名分的皇后”。

第二阶段：旧情入局。姚玲儿的表兄妹旧情、柳仪芳的替影处境、江晚晚的守密关系依次浮出。玩家开始意识到，中宫不是简单压服六宫，而是要决定谁能被保住、谁必须被敲打、谁可以成为盟友。

第三阶段：六宫决断。选秀、争宠、宫斗、怀孕、皇嗣和协理六宫事务开始互相牵连。玩家需要决定后宫秩序是宽厚、铁腕、平衡还是独断。

第四阶段：权力共主。若玩家在声望、位分、皇帝信任、皇嗣和朝局中取得足够支撑，她可以走向摄政、共主、独宠皇后或中宫孤权等结局方向。

### 1.5 结局方向

- `白头善果`：夫妻情分与中宫秩序都维持住，玩家仍是皇后，也是容安真正信任的人。
- `中宫共治`：情爱退居其次，玩家以皇后身份进入权力核心，成为后宫与前朝之间不可绕过的人。
- `独宠成牢`：玩家获得极高宠爱，却被所有权力和妃嫔关系反噬，尊贵反成囚笼。
- `六宫之主`：玩家不依附皇帝情爱，以制度、威望、皇嗣和盟友压住全局。
- `凤袍生寒`：玩家保住名分，却失去情分、盟友或关键皇嗣，成为体面但孤立的皇后。

## 2. 浮生如梦

### 2.1 路线定位

`浮生如梦` 是标准低位成长线，也是最接近普通玩家初次入宫体验的路线。玩家不带明确复仇任务，也没有皇后名分或和亲身份的强外部目标。她从储秀宫或低位起步，在日复一日的行动、交际、争宠和选择里，被皇帝、旧友和后宫逐渐看见。

这条线的核心问题是：一个普通入宫的人，如何在后宫里保住自己、保住重要的人，并决定自己最终想成为什么样的人。

### 2.2 故事开端

玩家宁小满入宫时，身份和未来都不确定。她可能出身不高，也可能只是众多新入宫女子之一。宫墙里没有人天然替她安排大义目标，她必须靠行动争取资源、宠爱、声望和关系。

沈妙清是这条线的情感锚点。她因玩家而入宫，对玩家天然偏向，性格清冷护短、寡言认死理。她不是主动撩拨型角色，而是会在沉默里替玩家承担压力的人。年欣兰则是路线压力来源：嘴硬、嫉妒、试探、不服软，常拿规矩压人，却不承担致命陷害一类不可逆恶事。

### 2.3 主要冲突

第一层冲突是“低位被看见”。玩家需要通过日常行动、才艺、宫宴、偶遇和侍寝逐渐得到容安注意。容安起初只把玩家视作后宫新人，随着事件推进，才会在她身上看见不同于普通妃嫔的选择和韧性。

第二层冲突是“旧友互保”。沈妙清对玩家有天然偏向，但这种偏向不是绝对安全。玩家若总让她挡在前面，她会受伤、被卷入流言，甚至被迫成长。玩家必须选择保护她的纯真，还是把她培养成能并肩承担后宫压力的盟友。

第三层冲突是“年欣兰的试探”。年欣兰表面挑刺，实际在玩家和沈妙清身上看见自己与旧友的影子。她的刁难本质是嫉妒与确认：她要看玩家会不会抛下沈妙清，会不会为了上位牺牲旧情。若二人经得住考验，年欣兰后期可以和解。

第四层冲突是“普通人生的选择”。这条线没有强制玩家必须复仇或夺权。玩家可以追求宠爱、友情、位分、安稳、皇嗣、权力，也可以在后期决定自己是否要继续留在这套规则里。

### 2.4 故事阶段

第一阶段：低位求存。玩家学习宫规、安排培养、积累声望与宠爱，沈妙清在旁边提供情感支撑，年欣兰制造压力。

第二阶段：旧友受伤。玩家的选择会影响沈妙清是否被卷入流言、宫斗或规矩惩戒。玩家第一次明确意识到，后宫里不是只有自己会承担后果。

第三阶段：被皇帝看见。容安通过偶遇、求情、宫宴或御书房事件，开始不再只把玩家当作普通妃嫔。他会显露疲惫，也会把某些选择权交给玩家。

第四阶段：三人关系转向。沈妙清从依赖或护短转向并肩，年欣兰从试探转向和解或继续敌对。玩家选择自己的后宫生存方式：温和经营、情感互保、争宠上位或权力扩张。

### 2.5 结局方向

- `浮生有归`：玩家在后宫中获得稳定位置，也保住沈妙清与自己的本心。
- `并肩长明`：玩家与沈妙清完成互保和共同成长，二人关系成为长期支撑。
- `旧友离心`：玩家为上位反复牺牲沈妙清，最终保住位分却失去旧友。
- `欣兰和解`：玩家经受住年欣兰试探，后期形成新的后宫平衡。
- `梦醒宫深`：玩家得到宠爱或位分，却发现自己已被后宫规则改变。

## 3. 影落掖庭

### 3.1 路线定位

`影落掖庭` 是低资源翻案线。玩家沉璧是罪臣之后，开局脱离掖庭罪役名籍后暂居储秀宫西偏殿；掖庭院保留为旧案、宫人差役与杂务地点。玩家银两、声望、宠爱都低，背负旧案污名。路线通关要求是十年内到达妃位，并达成结局【沉冤得雪】。

这条线的核心问题是：一个被判定不该再开口的人，如何取得说话资格，把被封住的旧案重新摆到御前。

### 3.2 故事开端

沉璧入宫时，宫中对她的公开认知只有“罪臣之后”。她的家族沈氏三年前被定为涉逆，案卷中有认罪供状、旧印与毒方。沈氏家眷或死或流放，她被投入掖庭，成为人人可以轻慢和试探的人。

但旧案并不干净。冷宫纸角、太医院旧方、长春宫旧库、陈氏旧印，都指向同一个事实：沈氏旧案可能是前朝党争与后宫封口共同造成的死案。

### 3.3 主要冲突

第一层冲突是“低位与真相”。玩家即便握有线索，也未必有资格递到御前。她必须先活下来，积累声望、宠爱、位分和可信度。

第二层冲突是“陈婉宁的封口”。陈婉宁是影落掖庭核心对手兼翻案枢纽。她端方、温柔、得体，会照拂玩家，却也知道旧案真相和陈家牵连。玩家不翻案时，她不会主动加害；玩家逼近旧案时，她会试探、交易、封口。只有高信任或高倾情下，她才可能背离母家。

第三层冲突是“容安是否重查”。皇帝最初并不真正相信玩家。他要判断玩家是在陈证旧案，还是借旧案攀附。玩家必须用证据、位分和可信度让他重新打开旧案。

第四层冲突是“手段的代价”。玩家可以宫斗、交易、偷取、威逼、攻略陈婉宁，但高宫斗污点会降低最终翻案可信度。真相不是只靠证据，也不是只靠宠爱，而是证据、身份、信任与代价共同结算。

### 3.4 故事阶段

第一阶段：掖庭求生。玩家建立旧案目标，学习藏锋或争取资源，第一次意识到自己必须先获得活下去的资格。

第二阶段：旧案浮出。陈婉宁笑里藏刀地试探，冷宫旧闻和太医院旧方逐步证明案卷存在问题。

第三阶段：交易与证物。陈婉宁提出封口交易，玩家可以接受、拒绝或假意周旋。随后证物匣出现，伪印供状与旧方成为翻案关键。

第四阶段：御前质问。玩家以足够证据和身份进入养心殿，面对容安对动机与证据的双重质问。

第五阶段：最终阻击或协助。陈婉宁根据关系与玩家行为，选择反咬封口或暗中递出陈氏旧印。最终进入重审，或被封存、冷宫、苟活等偏结局吞没。

### 3.5 结局方向

- `沉冤得雪`：玩家到达妃位，证据充足，容安同意重审，沈氏旧案被推翻。
- `旧案封存`：证据或皇帝信任不足，旧案重新封档。
- `身陷冷宫`：翻案失败且被反咬，玩家成为新的旧事。
- `权势苟活`：玩家爬上高位，却主动放弃翻案。
- `孤证难鸣`：证据足够，但位分与声望不足，真相无人听见。

## 4. 尘缘夙错

### 4.1 路线定位

`尘缘夙错` 是和亲、异族、故国与真心冲突线。玩家乌兰托娅是和亲公主，入宫居玉清宫，带着母国和书，也带着对故国、自由与旧人的牵挂。她在宫中既是妃嫔，也是政治符号。

这条线的核心问题是：玩家要成为大景后宫中的异邦宠妃、连接两国的权力人物，还是选择故国旧情与离宫自由。

### 4.2 故事开端

玩家跨过宫门时，身份已经被写入和书。宫里看她，先看她的来处、礼数、血脉和政治价值。她拥有较高初始属性与声望，却承受更强的凝视：她的一言一行都可能被解读成故国态度。

容安起初把她当成和书的一部分，不完全把她当作独立的人。杜云嫣则在宫中靠近她，娇气病弱、依赖体贴，喜欢听草原故事。阿翎是路线专属关键人物，来自草原，是玩家真正牵挂的人。阿翎不属于后宫妃嫔池，不参与位分、宠爱和声望体系，她代表宫外自由、故国旧情与离开皇城的可能。

### 4.3 主要冲突

第一层冲突是“和亲身份”。玩家的宠爱、皇嗣、晋升和立储资格都带有特殊限制。普通和亲公主或异国贡女所出子嗣默认不可立储，但尘缘夙错是特殊例外线，可以通过主线条件改写这一点。

第二层冲突是“容安的认知变化”。他起初把玩家当成和书、筹码、异邦象征。随着相处，他才逐渐把玩家当成有痛苦、有选择、有欲望的活人。

第三层冲突是“杜云嫣的成全”。杜云嫣前期依赖玩家，喜欢听草原故事；中期转为心疼与不舍；后期知道阿翎后，会鼓励玩家离开并帮忙打掩护。她的推进顺序固定是依赖、亲近、心疼、不舍、成全，不能跳段。

第四层冲突是“阿翎与离宫”。阿翎第 3 个月宫门重逢，后续通过离宫邀请、年度书信、第 4 年夜闯宫门等节点持续拉扯玩家。她不高频出场，而是以记忆、旧物、消息和关键抉择保持分量。

第五层冲突是“故国与皇位”。玩家可以选择皇帝的爱、皇后的权力、故国利益、阿翎旧情或离宫自由。每条路都意味着失去另一部分。

### 4.4 故事阶段

第一阶段：和亲入宫。玩家熟悉玉清宫、宫门、养心殿、御书房等关键地点，明白自己不是普通妃嫔，而是被两国共同注视的政治人物。

第二阶段：宫门重逢。第 3 个月，阿翎出现在宫门。她带来故国气息，也第一次让玩家意识到：宫外还有人记得她不是和书上的名字。

第三阶段：宫中牵绊。容安开始从政治符号中看见玩家本人；杜云嫣依赖玩家并逐渐心疼她；玩家在宠爱、礼法、故国消息和宫中生存之间摇摆。

第四阶段：年度书信。若玩家拒绝离宫，阿翎会留下并通过每年 12 月清晨的来信延续存在。第 3 年，她会坦白自己自幼倾心于玩家，并表达想成为女将军保护玩家的心意。

第五阶段：第 4 年大分支。阿翎夜闯宫门，邀玩家离开。玩家若拒绝，阿翎可能死亡；若接受，进入离宫方向；若关系与政治条件特殊，可能进入阿翎成为将军、与容安开战、战死或以书信劝容安善待玩家等分支。

第六阶段：终局抉择。玩家决定自己要皇帝的爱、皇后线与立储例外、故国权力、阿翎旧情，还是离开皇城。尘缘夙错的结局不应只有爱情，也要回收和亲身份、血脉政治和故国命运。

### 4.5 结局方向

- `尘缘归塞`：玩家选择离宫，回到故国或草原旧路，与阿翎线完成回收。
- `宫墙夙错`：玩家留在宫中，失去阿翎或故国旧情，成为大景后宫中最华丽的异类。
- `异邦中宫`：玩家突破和亲身份限制，进入皇后线，以政治能力改写血脉偏见。
- `两国刀兵`：故国、阿翎与容安的矛盾失控，玩家被迫承受战争结果。
- `绝代红颜`：玩家以宠爱与政治价值立足，却始终被当作两国之间的象征。
- `来信未绝`：玩家拒绝离宫，但阿翎以书信与远方功业继续存在，成为宫中岁月里的长期牵挂。
