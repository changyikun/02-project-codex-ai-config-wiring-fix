# 宫斗事务硬规则（架构目录版）

## 一、文档定位

本文件定义玩家与 NPC 共用的宫斗事务规则。  
系统负责真实数值，AI 只负责：

- 娇娇的通报
- 养心殿对话
- 与被害妃嫔、陷害妃嫔的后续剧情

## 二、宫斗行为类型

### 2.1 造谣

- 欺凌宫人：轻
- 奢侈浪费：轻
- 与人偷情：中
- 不敬先祖：重
- 意图谋逆：重

### 2.2 下毒

- 陨颜丹：轻
- 麝香：中
- 鹤顶红：重

## 三、时间流程

- 白天选择宫斗行为
- 当旬夜晚统一结算
- 下一旬由娇娇通报结果
- 若暴露，则进入最多 `3` 旬调查
- 玩家相关重案会被带往养心殿

## 四、两次检定

每次宫斗固定两次检定：

1. 行动检定：决定行为是否生效
2. 隐匿检定：决定是否暴露并进入调查

四种标准结果：

- 生效且未暴露
- 生效但暴露
- 未生效但未暴露
- 未生效且暴露

## 五、基础公式

### 5.1 攻击值

- 造谣：`A = 发动者心计 / 10`
- 下毒：`A = 发动者药理`

### 5.2 防御值

- 造谣：`D = 目标心计 / 10 + 目标宠爱 / 20`
- 下毒：`D = 目标心计 / 10 + 目标宠爱 / 20 + 目标药理 / 4`

### 5.3 行动成功率

```txt
行动成功率 = 50 + A - D - 事件修正 - 嫁祸修正
```

- 轻：`0`
- 中：`5`
- 重：`10`
- 嫁祸：额外 `12`
- 最终截断：`10~90`

### 5.4 隐匿成功率

```txt
隐匿成功率 =
50
+ 发动者心计 / 12
- 目标心计 / 15
+ 发动者宠爱 / 8
+ 合谋修正
- 事件修正
- 嫁祸修正
```

- 轻：`5`
- 中：`10`
- 重：`15`
- 合谋：`+5`
- 嫁祸：额外 `15`
- 最终截断：`5~85`

## 六、合谋、嫁祸与福德

### 6.1 合谋

- 条件：好感 `> 60`
- 合谋后攻击值：

```txt
floor((发动者攻击值 + 合谋者对应攻击值) / 2) + 10
```

- 隐匿 `+5`
- 案件成立时初始获罪率 `+10`

### 6.2 嫁祸

- 行动检定 `-12`
- 隐匿检定 `-15`
- 初始获罪率 `+15`
- 调查期间每旬额外 `+5`

### 6.3 主动宫斗对福德的影响

- 主动造谣：福德 `-5`
- 主动下毒：福德 `-10`

## 七、调查与获罪率

### 7.1 进入调查

隐匿检定失败即进入调查。

### 7.2 初始获罪率

```txt
初始获罪率 =
基础值
+ (100 - 隐匿成功率) / 2
+ max(0, 目标宠爱 - 20) / 2
+ 嫁祸值
+ 合谋值
- 发动者宠爱 / 5
```

基础值：

- 轻：`20`
- 中：`40`
- 重：`65`

### 7.3 每旬增长

- 轻：`+8`
- 中：`+14`
- 重：`+20`
- 嫁祸额外：`+5`

### 7.4 三旬结案

```txt
最终获罪率 = 初始获罪率 + 三旬增长 - 打点减值
```

- `< 100`：疑案
- `>= 100`：定罪

## 八、个人事务中的干预

当玩家自己是案件对象时，干预统一在“个人事务”中处理。

### 8.1 压低自身嫌疑

- `20` 银两：玩家自身嫌疑 `-5`

### 8.2 推高陷害妃嫔嫌疑

若本案已锁定一名陷害妃嫔，则玩家可改为：

- `20` 银两：陷害妃嫔嫌疑 `+5`

若该妃嫔嫌疑值到达 `100`，则该妃嫔立即定罪。

### 8.3 干预次数

个人事务中的银两干预不设固定次数上限。  
只要案件未结案、玩家仍有银两，就可以继续操作。

### 8.4 未锁定陷害者时的规则

若本案没有明确锁定陷害妃嫔，则玩家只能压低自己的嫌疑，不能平空指定他人背锅。

## 九、旁观案件

若案件与玩家无直接嫌疑关系，纪事仍显示嫌疑人列表。  
玩家可对任一嫌疑人反复操作：

- `20` 银两：嫌疑 `-5`
- `20` 银两：嫌疑 `+5`

## 十、NPC 互斗

NPC 妃嫔之间也会使用同一套宫斗公式。  
触发前提：

- 双方交恶
- 其中一方野心 `> 50`
- 每旬该进攻方有 `10%` 概率发动一次宫斗

每个符合条件的 NPC 每旬最多只发动 `1` 次主动宫斗，避免后宫事件量爆炸。

## 十一、养心殿

以下情况会把玩家带到养心殿：

- 玩家主动宫斗后暴露
- 玩家被他人陷害，需要自证
- 玩家是被害妃嫔，且皇帝亲自过问此案

养心殿只负责：

- 自证
- 求情
- 对最终惩罚做有限修正

## 十二、适合写入总架构目录的统一语义

- 宫斗固定分为造谣与下毒两类
- 每次宫斗固定进行“行动检定 + 隐匿检定”
- 主动造谣扣福德 `5`，主动下毒扣福德 `10`
- 调查期固定最多 `3` 旬
- 个人事务中的案件干预统一改为 `20 银两 -> 5 点嫌疑变化`
- 陷害者存在时，玩家可持续加高其嫌疑直到其被定罪
- 旁观案件同样允许玩家用银两上下调嫌疑值
- NPC 之间每旬也可能互斗，形成持续宫斗氛围
