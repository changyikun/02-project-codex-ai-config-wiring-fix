# UI元素需求清单

## 主菜单
- 四条可选路线按钮：兰因絮果、浮生如梦、影落掖庭、尘缘夙错
- 一条禁用按钮：未完待续
- 16:9 横屏舞台

## 属性加点
- 姓名输入
- 家世选择
- 年龄输入
- 5项主属性：健康、福德、心计、容貌、气质
- 5项副属性：才艺、丹青、女红、药理、政治
- 剩余点数显示
- 自动分配 / 确认入宫按钮

## 剧情简报
- 120-180字背景故事
- 冲突伏笔
- 进入剧情按钮

## Gal 对话
- 左右立绘占位
- 底部对话框
- 每回合 3 个不重复选项
- 寝殿 / 地图切换按钮

## 寝殿活动
- 左侧妃嫔状态面板：位分、好感、势力值
- 活动按钮列表
- 行动点与压力显示

## 皇宫地图
- 可点击区域：御书房、御膳房、建章宫、宣武门、宫门、太医院、妙音堂、宝华殿、重华宫、椒房殿、储秀宫、长春宫、启祥宫、钟粹宫、昭阳宫、玉清宫、永宁宫、永和宫、延禧宫、临华殿、昭华殿、披香殿
- 区域事件文本显示区
