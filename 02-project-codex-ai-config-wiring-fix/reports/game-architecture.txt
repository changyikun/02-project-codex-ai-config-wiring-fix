2.2 成长系统

核心逻辑：行动——属性——触发剧情——状态变化/获得物品

角色基本属性

声望（-2000~5000）口角，宴会才艺成功，侍寝成功，妃子美言或诋毁

宠爱（-100~100）依靠与皇帝的有效互动，皇帝心情，是否有其余妃子争宠

福德（-100~100）是否有陷害，造谣，下毒。礼佛会增加福德

野心（-100~100）取决于角色宫斗行为的频率与倾向性

心机（0~1000）心机越高，宫斗时胜算越大。他人下毒越容易发现

容貌（0~1000）可通过梳妆、保养、特殊剧情与高阶装扮提升

气质（0~1000）可通过读书，书画，乐理来增加

健康（0~1000）可通过服用补品来增加

压力（0~100）宫斗频率越高压力越大 玩家的压力值大于80会开始减寿命 每月减去0.2。*默认寿命为60岁~80岁之间开局随机寿命。寿命为隐藏值。可在玩家花钱算卦占卜时看见数字。

压力值来源于宫斗杀害的人太多，第一次杀人默认加20压力值，第二次10第三次5，大于3人后每次默认加5压力值，陷害或者造谣他人默认加3。

妃子压力值大于80会导致性情大变（包括但不限于野心值，心计值，性格由温柔变清冷等等的变化）并有概率疯癫。

1.贵妃：姚玲儿

【角色定位】

19岁。三品文官嫡女。贵妃。与皇帝是青梅竹马的表兄妹。

她不是纯粹恶人，而是“高位、旧情、强自尊”三者叠加后形成的危险对手。

她真正的核心不是单纯争宠，而是：

- 认为自己本该比玩家更有资格接近皇帝
- 极度在意脸面，不接受被低位妃嫔踩踏
- 对皇帝仍有旧情，但这份旧情里混着不甘、占有欲与胜负心
- 只要玩家给过她足够体面与实利，她并非不能合作，甚至可以转为权力同盟

因此她必须被设计成：

- 前期强竞争者
- 中期可拉拢对象
- 后期可能成为盟友，也可能成为最狠的反扑者

【初始公开状态】

- 姓名：姚玲儿
- 年龄：19
- 位分：贵妃
- 居所：长春宫
- 家世：三品文官嫡女
- 公开性格：骄矜、好胜、嘴上不饶人、极重脸面
- 对玩家表层态度：不服中宫，但尚未彻底撕破脸

【初始数值建议】

- 声望：`2180`
- 宠爱：`72`
- 健康：`720`
- 压力：`32`
- 福德：`8`
- 野心：`48`
- 心计：`780`
- 容貌：`860`
- 气质：`780`
- 家世权重：`68`
- 对玩家初始好感：`-20`
- 对玩家初始倾情：`0`
- 子嗣：`0`

【初始隐藏状态建议】

- 对皇帝真心值：`70`
- 对玩家敌意基线：`35`
- 黑化起点：`0`
- 是否可攻略：是
- 早期关键词：旧情、嫉妒、体面、高位压人
- 后期关键词：结盟、夺权、醋意、反扑

【角色合理性说明】

她必须比普通高位妃更强，否则“贵妃 + 青梅旧情”失去威胁感。

但她也不能一开始强到压死玩家，否则皇后线没有操作空间。

所以当前建议是：

- 她的初始宠爱高于大部分妃子，但不直接高到独宠
- 她的对玩家态度明确偏负，但还没到不死不休
- 她的高压来自“地位 + 旧情 + 自尊”，而不靠无脑发疯

【固定故事节点 1：欺凌宫人谣】

这是姚玲儿的强制早期节点，必须加入。

【触发条件】

- 线路为兰因絮果
- 姚玲儿存活，且当前位分为贵妃
- 后宫中存在一名随机妃子，满足 `宠爱 < 30`
- 该随机妃子状态为正常存活，非冷宫，非已逝
- 玩家进入大地图后，在 `后宫 / 御花园` 场景的 `下午 / 傍晚` 有一次撞见密谋的机会
- 本节点未触发过

【事件表现】

玩家在大地图中撞见该随机妃子的宫女与外头小太监串话，准备散播“姚玲儿纵容宫人欺压下人、私下掌掴宫女”的流言。

重点不是现场抓到姚玲儿欺人，而是抓到“有人要借此做局”。

【玩家选项】

  - 姚玲儿对玩家好感 `+15`

  - 姚玲儿保住贵妃位
  - 获得旗标：`yao_helped_rumor_case = true`
  - 获得旗标：`yao_owes_player_once = true`

2. 不帮助

- 结果：
  - 皇帝借“贵妃约束宫人不严”为由，将姚玲儿由贵妃降为妃
  - 姚玲儿对玩家好感 `-20`
  - 姚玲儿敌意 `+15`
  - 姚玲儿宠爱 `-12`
  - 姚玲儿声望按降位规则回落到妃位区间
  - 获得旗标：`yao_demoted_by_rumor = true`

3. 帮助造谣者
- 玩家选择替造谣者遮掩，或反向坐实谣言。
- 结果：
  - 皇帝借此将姚玲儿由贵妃降为妃
  - 姚玲儿对玩家好感 `-35`
  - 姚玲儿敌意 `+25`

【固定故事节点 5：不舍却成全】

这是杜云嫣最重要的成长节点。

【触发条件】

- `du_knows_aling = true`
- 杜云嫣对玩家好感、倾情、依赖均达到较高区间
- 玩家明确表达过“自己想离开”

【事件核心】

杜云嫣会非常不舍。

因为她其实不想让玩家走。

她会羡慕玩家：

- 羡慕玩家至少还知道自己想去哪里
- 羡慕玩家心里还有一个阿翎
- 羡慕玩家哪怕被困住，也还有想奔回去的方向

但正因为她喜欢玩家，她会选择成全，而不是挽留。

她会说出的态度应该是：

- 我不想你走
- 可若你留下来会更难过，那我宁可帮你

【结果】

- 获得旗标：`du_encourages_escape = true`
- 杜云嫣倾情进一步提升
- 她正式从“依赖你的人”转成“愿意成全你的人”

【固定故事节点 6：替你打掩护】

【触发条件】

【事件核心】

杜云嫣会利用自己的病弱身份和娇气性格制造遮掩空间，例如：

- 借自己常来缠玩家的习惯，为玩家寝殿的异常动静打圆场

重点是：

- 她不会亲自跑去冒险做不符合她体质和身份的事
- 她的掩护方式必须符合她“病弱娇贵”的人物逻辑

【结果】

系统补充规则（新增硬规则整理）

一、经济与银两

1. 玩家每月按位分领取月俸。若位分低于妃且宠爱低于20，则月俸固定克扣：

- 宠爱1~19：按70%发放
- 宠爱<=0：按50%发放
- 冷宫中：月俸为0

2. 若玩家宠爱>=61，则当月实际月俸+20%。

3. 每月最多2名倾情>60的妃子会将自己当月实际月俸的1/4赠送给玩家。该赠银总额允许高于玩家基础月俸。

4. 玩家每月固定结算宫务开销，按当月基础月俸 `W` 计算：

- 基础宫务 = `floor(W * 0.2)`
- 宠爱>=61时，高宠额外宫务 = `floor(W * 0.1)`

高宠额外宫务不得高于高宠额外收入，因此高宠路线始终为净收益。

5. 银两的大额消耗池需要常驻系统，包括：

- 朝臣打点
- 家族谋划
- 子嗣养育与礼仪
- 稀有礼物与补品
- 宫务整饬
- 宫斗案件运作

6. 政治与银两挂钩：

- 政治<20：不可正式朝臣打点
- 政治20~39：180银两=朝臣支持+2，320银两=+4
- 政治40~59：180银两=+3，320银两=+5，500银两=+8
- 政治60+：180银两=+3，320银两=+6，500银两=+10

7. 家族谋划当前按季度结算，仅保留第一档：120 银两。

二、旁观案件的银两干预

1. 若宫中发生调查案件且与玩家无直接嫌疑关系，则纪事中显示其他妃子的嫌疑值。

2. 玩家可花费 20 银两对某名嫌疑人进行一次干预：

- 20银两：嫌疑值 -5
- 20银两：嫌疑值 +5

- 触发定位：
  情感转折节点。
- 触发条件：
  完成至少 `2` 次 `草原夜话`
- 事件表现：
  她从“想听新鲜故事”，转为开始听懂玩家说故事时藏着的失落、想念和压抑。
  她会第一次主动安慰玩家，给玩家留药、留暖手炉、留香囊，或轻声说“你讲这些的时候，好像并不开心”。
- 数值结果：
  好感 `+10`
  倾情 `+8`
  开启旗标 `du_feels_for_player`
- 节点作用：
  这是她从索取情绪价值，转为愿意回头照顾玩家的关键一步。

`阿翎之事`
- 触发定位：
  核心知情节点。
- 触发条件：
  `du_feels_for_player = true`
  且玩家已触发阿翎相关主线信息
- 事件表现：
  杜云嫣会逐步意识到，玩家反复提起的草原并不是抽象远方，而是一个具体的人与一段没有断掉的牵挂。
  她不会立刻吃醋发作，而是先安静下来，先听懂，再难过。
- 玩家分支：
  如实告诉她阿翎的存在与意义：好感 `+12`，倾情 `+10`，开启 `du_knows_aling`
  只说一半：好感 `+4`，后续仍可推进
  强行隐瞒：推进冻结
- 节点作用：
  她的成熟度在这里体现出来。她不是因为知道玩家心里有人就翻脸，而是会先替玩家心疼。

`不舍却成全`
- 触发定位：
  高情感节点。
- 触发条件：
  `du_knows_aling = true`
  杜云嫣好感 `>= 68`
  杜云嫣倾情 `>= 42`
- 事件表现：
  她会在私下和玩家谈一次真正的心。她承认自己不想让玩家走，也承认自己羡慕阿翎，羡慕有人能被玩家这样惦记。
  但她最后仍会对玩家说：如果你真的决定走，就该去你想去的地方。
- 数值结果：
  好感 `+10`
  倾情 `+12`
  开启旗标 `du_supports_escape`
- 节点作用：
  这里必须写出“舍不得”和“成全”并存，不能只写成体贴，少了痛感就不成立。

`替你打掩护`
- 触发定位：
  行动协助节点。
- 触发条件：
  `du_supports_escape = true`
  且玩家已进入明确离宫准备阶段
- 事件表现：
  杜云嫣会用自己病中召医、夜里服药、宫人换班、寝宫闭门静养等方式替玩家遮掩行动痕迹。
  她不会参与粗暴执行，但会提供极关键的安静窗口。
- 玩家分支：
  接受她的帮助：开启 `du_cover_escape`
  拒绝牵连她：杜云嫣好感 `+6`，仍保留协助资格
- 节点作用：
  她的帮助不是因为她不在乎，而正是因为她在乎。

  - 定位：宫门处固定商贩 NPC
  - 用途：物品售卖、物品回收、宫门交易入口承接

2.杜娘回收硬规则
- 杜娘支持回收可售卖物品。
- 回收价格 = 该物品标准售卖价格的 `80%`
- 计算公式建议写死为：
  `recycle_price = floor(shop_price * 0.8)`
- 若该物品未定义 `shop_price`，或被标记为不可售卖，则默认不可回收。
- 回收行为不额外消耗体力，不推进独立剧情时段，默认属于交易类即时结算行为。

3.接入建议
- 物品表后续建议新增字段：
  - `can_sell`
  - `shop_price`
  - `can_recycle`
  - `recycle_price_override`
- 默认规则：
  - `can_sell = true` 的物品，若未单独禁用，则 `can_recycle = true`
  - 未设置 `recycle_price_override` 时，自动按 `shop_price * 0.8` 向下取整
- 这样后面商店接入时，杜娘不需要单独再手写每个物品的回收价。
